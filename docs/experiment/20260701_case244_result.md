# case244 result: task205 all-ones Conv to AveragePool

## 結果
- 対象: `data/output/onnx/task205.onnx`
- baseline: `cost=12854` (`params=640`, `memory=12214`, `n_pass=266`, `n_fail=0`)
- 変更後: `cost=12494` (`params=280`, `memory=12214`, `n_pass=266`, `n_fail=0`)
- local gain: `ln(12854 / 12494) = 0.0284`
- Kaggle submit: `54211698`
- Public Score: `7181.26 -> 7181.28`
- 採否: 採用

## 変更内容
- 先頭の `Conv(input, k6)` を `AveragePool(input)` に置換。
- `k6` は全1の `(10,1,6,6)` float32 depthwise kernel で、後段は `ReduceMax` / `ArgMax` による channel ranking のみ。
- `AveragePool(kernel_shape=[6,6], strides=[8,8])` は sum の `1/36` 倍だが、全 channel 同一スケールなので ranking は不変。
- 未使用になった `k6` initializer を削除。

## 結論
全1 depthwise Conv は、後段用途が channel argmax に限定されるため AveragePool で等価に置換できた。memory は維持しつつ params を360削減し、実 LB でも `+0.02` 表示改善を確認できたため採用する。
