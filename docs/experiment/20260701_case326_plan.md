## Hypothesis

The tail currently creates a 9-channel bool tensor with seven explicit zero
channels, then pads to the final output:

```text
Concat(bg, zero6, zero6, zero6, zero6, zero6, zero6, zero6, diag) -> compact
Pad(compact, pads) -> output
```

Replace this with a two-channel packed live tensor, cast it to uint8, and use a
one-hot `QLinearConv` to restore the 10-channel layout while folding the final
spatial pad:

```text
Concat(bg, diag) -> compact_packed
Cast(compact_packed) -> compact_packed_u8
QLinearConv(compact_packed_u8, one-hot 2ch->10ch weights, pads=final output pad)
  -> output
```

The projection maps output channel 0 from `bg` and output channel 8 from
`diag`.
