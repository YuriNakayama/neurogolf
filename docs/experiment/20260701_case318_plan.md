```text

Pad(out20, sp=[0,0,0,0,0,4,10,10], value=0) -> output
```

Only three live channels are needed. Replace the tail with:

```text

QLinearConv(packed3, one-hot 3ch->10ch weights, pads=[0,0,10,10]) -> output
```

The one-hot projection maps:

```text

output ch4 <- mask20
output ch5 <- c5u
all other output channels <- 0
```

and performs the bottom/right zero pad in the same node. This deletes the
`out20` 6-channel intermediate and avoids materializing explicit zero channels.
