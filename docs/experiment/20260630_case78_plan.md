# 20260630 case78 plan — t319 の global input_u8 Cast を局所 plane Cast へ置換できるか検証

## 仮説
t319 は cost 22097（params 269 / memory 21828）で、先頭に `Cast input -> input_u8`
（`[1,10,30,30]` uint8 = 9000B）を持つ。実際に必要なのは active 3色の object plane と
occupancy のため、全10chを uint8 化せず、3つの object plane だけを Gather 後に Cast すれば
memory を削れる可能性がある。

## 対象
- task319
- 現行: cost 22097, params 269, memory 21828, n_fail 0

## 手法
1. `ReduceMax(input_u8)` の occupancy 計算を `ReduceMax(input)` に変更。
2. `Gather(input_u8, colorX)` を `Gather(input, colorX) -> Cast(uint8)` に変更。
3. `audit_one(run_correctness=True)` で faithful と cost を確認。
