## Hypothesis

The tail currently builds a sparse 6-channel bool strip, repeats it across
width, and pads to the final grid:

```text
Concat(black_col, zero_col, zero_col, zero_col, zero_col, solid_col) -> top6_col
Concat(top6_col, top6_col, top6_col) -> top6
Pad(top6, pads) -> output
```

Replace it with only the two live bool channels, repeat the packed strip, cast
to uint8, and project back to the final 10-channel output:

```text
Concat(black_col, solid_col) -> col2
Concat(col2, col2, col2) -> top2
Cast(top2) -> top2_u8
QLinearConv(top2_u8, one-hot 2ch->10ch weights, final pads) -> output
```

The projection maps the packed channels back to output channels 0 and 5.
