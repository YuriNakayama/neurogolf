## Hypothesis

The tail currently builds a 6-channel 10x10 active tensor with two explicit zero
planes, then pads to the 10-channel 30x30 output:

```text
Concat(out0, zero_plane, out2, zero_plane, zero_plane, gray_u8) -> active
Pad(active, outpad, value=0) -> output
```

Only three live channels are needed. Replace this with:

```text

QLinearConv(packed_tail, one-hot 3ch->10ch weights, pads=[0,0,20,20]) -> output
```

The projection maps:

```text

output ch5 <- gray_u8
all other output channels <- 0
```

and performs the bottom/right zero pad in the same node.
