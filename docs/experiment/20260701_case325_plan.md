## Hypothesis

The tail currently builds a 9-channel bool tensor with six explicit zero
channels, then pads to the final output:

```text
Concat(ch0_b, zero_b, zero_b, zero_b, zero_b, zero_b, zero_b, ch7_b, ch8_b)
  -> output_9x11
Pad(output_9x11, pad_to_30) -> output
```

Replace this with a packed live-channel tensor, cast it to uint8, and project it
to the 10-channel output with one-hot `QLinearConv`:

```text
Concat(ch0_b, ch7_b, ch8_b) -> packed_live
Cast(packed_live) -> packed_live_u8
QLinearConv(packed_live_u8, one-hot 3ch->10ch weights, pads=final output pad)
  -> output
```

The graph output type is updated from bool to uint8, matching prior accepted
one-hot submissions and preserving scorer correctness.
