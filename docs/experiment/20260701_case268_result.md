## 結論

terminal で不要な float 中間を残している小〜中 cost task は、絶対削減量が小さくても
相対削減率で submit 価値がある。scorer が `out > 0` 判定を使う task では、出力 dtype を
float から uint8 に落とすだけで安全に memory cost を削れる場合がある。
