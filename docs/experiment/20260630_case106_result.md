対象:

```text
Where(sepmask, linecu, exp) -> withsep
Where(ingrid, withsep, sentu) -> fu
Equal(fu, lvu) -> output
```

| 変更 | cost | 結果 |

| `fu` bypass: `Equal(withsep, lvu)` | 17136 | `n_fail=231` |
