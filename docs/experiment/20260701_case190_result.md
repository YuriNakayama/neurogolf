## Analysis

The graph is already very compact:

```text
score = Conv(input, score_weights, score_bias)
keep = Cast(score -> bool)
output = Where(keep, input, bg)
```

The full-grid `score` and `keep` tensors are both semantically required. ONNX
does not allow the float `score` tensor as a `Where` condition, and constant
conditions collapse the task to all-input or all-background outputs, which fails
every public example.

## Conclusion

No exact cost-reducing valid ONNX was found. The baseline ONNX was unchanged and
no Kaggle submission was made.

## Next

Avoid simple `Conv -> Cast -> Where` graphs unless there is an existing bool
condition with the same shape. Continue searching for removable full-shape
intermediates with an already valid replacement tensor.
