## Background

case185 adopted the first removal rewrite in `task069`:

```text
rem1 = Where(Cast(t1_lbl), zero_u8, c8_u8)
```

became:

```text
rem1 = Sub(t1_lbl, c8_u8)
```

That change passed local audit and improved the real Public Score. The current
graph still has the same pattern one step later:

```text
rem2 = Where(t2_nz_b, zero_u8, rem1)
```

where:

```text
t2_nz_b = Cast(t2_lbl)
```

## Hypothesis

`t2_lbl` is a uint8 0/1 placement mask for the second extracted patch. At this
stage `rem1` is the remaining 8-channel mask after the first extraction. The
second placement is selected from the remaining mask, so subtracting `t2_lbl`
from `rem1` is equivalent to zeroing those cells:

```text
rem2 = Sub(rem1, t2_lbl)
```

This removes the bool cast tensor `t2_nz_b` and the `Where` branch while keeping
the same uint8 shape and static graph.
