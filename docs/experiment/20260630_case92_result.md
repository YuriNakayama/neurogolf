## 実施内容

t367 は 11 node の極小 graph。color 0/5 channel を 20x20 crop し、`rect_w`（shape `(25,2,9,9)`）で rectangle feature を作り、`fill_w`（shape `(1,25,5,5)`）で mask を作る。主コストは `rect_w` param と 20x20 中間。

### 重み構造

- `rect_w`: int8 `(25,2,9,9)`, nonzero 825
- `rect_b`: int32 `(25,)`
- `fill_w`: uint8 `(1,25,5,5)`, nonzero 225
- `fill_w` の入力 channel は `rect_w` の 25 output channel と一対一で対応している。

### channel zero probe

- `rect_w` output channel 0..24 をそれぞれ 0 固定:
  - 最良でも channel 19 の `n_fail=11`
  - 全 channel が `n_fail>0`
- `fill_w` input channel 0..24 をそれぞれ 0 固定:
  - `rect_w` channel zero と同じ failure pattern
  - 全 channel が `n_fail>0`

zero 化自体は params/memory/cost を下げないが、構造縮約可能な未使用 channel があるかの判定として実施。結果として全 channel が faithful 上必要。
