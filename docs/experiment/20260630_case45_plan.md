# 20260630 case45 plan — floor net 群の構造クラスタリングで一括削減の波及先を探す

## 仮説
複数タスクの floor net が同一構造（同一 weight shape の single-Conv 等）を共有していれば、
1つの少 param な等価表現で複数タスクに波及削減できる。特に in_channels を真の使用色数まで
削減する技法が未適用のタスクがあれば勝てる。

## 手法
- 全 single-Conv floor を weight shape でクラスタリング。
- in_channels が真の使用色数より大きいタスク（過剰channel）を特定 → channel削減 net で置換。

- faithful n_fail=0 かつ cost<floor かつ 実LB>7180.78 で採用。退行 revert。
