## Hypothesis

The tail currently creates a 6-channel bool tensor with three explicit zero
channels, then pads to the final output:

```text
Concat(bg_b, zero_b, block_b, zero_b, zero_b, og_b) -> grid6
Pad(grid6, pads, pv) -> output
```

Replace this with the three live channels and a one-hot `QLinearConv`
projection:

```text
Concat(bg_b, block_b, og_b) -> grid6_packed
Cast(grid6_packed) -> grid6_packed_u8
QLinearConv(grid6_packed_u8, one-hot 3ch->10ch weights, pads=final output pad)
  -> output
```

The projection maps live channels back to output channels 0, 2, and 5.
