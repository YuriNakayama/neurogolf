## 仮説
`task050` は channel 8 の crop から行/列の first/last endpoint を求め、水平・垂直 fill mask を作る。

現行 graph は last endpoint を次の形で求めている。

```text
rpos_f = MatMul(x8, ccol_f)
rpos = Cast(rpos_f)
rlast = Sub(rpos, rfirst)
```

同様に列方向も `MatMul(crow_f, x8)` で求めている。行/列に endpoint がある場合は `ArgMax(select_last_index=1)` で last を直接得られる可能性があり、座標 MatMul と float32/int8 中間を削れる見込み。

## 手法
1. `/tmp` で row/col の last endpoint を `ArgMax(select_last_index=1)` に置換する。
2. 空行・空列では ArgMax が最後のゼロ位置を返すため、必要なら `ReduceMax` presence guard で 0 に戻す。
3. `audit_one` で `n_fail=0` かつ cost 減なら本体に反映して submit する。
4. cost 悪化または fail が出る場合は提出しない。
