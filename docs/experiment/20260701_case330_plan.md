## Hypothesis

The tail currently builds a sparse 10-channel patch with eight explicit zero
channels and pads it to the final output:

```text
Concat(black, zero, zero, zero, zero, gray, zero, zero, zero, zero) -> patch
Pad(patch, out_pads) -> output
```

Replace it with the two live uint8 channels and a one-hot `QLinearConv`
projection:

```text
Concat(black, gray) -> patch_packed
QLinearConv(patch_packed, one-hot 2ch->10ch weights, final pads) -> output
```

The projection maps the packed channels back to output channels 0 and 5.
