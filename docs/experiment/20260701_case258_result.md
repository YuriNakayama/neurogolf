## 結論

zero cell が slice 内にあり、guard なしでは Conv の 0 出力を再現できなかった。guard を足すと完全一致するが、`ArgMax` の int64 中間と guard 分が重く、baseline より cost が増える。ONNX は変更せず submit しない。

## 次の一手
