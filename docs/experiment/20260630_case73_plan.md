# 20260630 case73 plan — bool復帰 Cast の dtype 拡張候補を検証する

## 仮説
case72 では `Cast(bool)->Cast(uint8)` の復帰だけを検証したが、同じ構造で
`Cast(bool)->Cast(float16/int32)` に戻す候補もある。元テンソルが既に適切な 0/1 値なら、
復帰 Cast を消して memory を削れる可能性がある。

## 対象
- `Cast(bool)->Cast(float16/int32/uint8)` の全候補16件
- case72 未検証の主対象: t148, t280, t316

## 手法
1. 復帰 Cast の使用先を元テンソルへ置換する probe ONNX を生成。
2. `audit_one(run_correctness=True)` で faithful 検証。
3. n_fail=0 かつ cost 減の候補のみ採用・submit。
