The graph is already compact:

```text
bandc = Where(isvert, colcolor, rowcolor)
hcell = Where(iszero, bandc, 0)
hb    = Where(isvert, ReduceMax(hcell, width), ReduceMax(hcell, height))

cg    = Where(ingrid, cg0, sent)
```

Replacing `hb = Where(...)` with orientation-specific `Equal` branches would
remove `hb`, but it would materialize extra full-grid `mark` candidates before
the scalar branch. Fusing `cg0` and `cg` similarly adds a full-grid combined
condition or nested target, so it does not create a narrow cost reduction.
