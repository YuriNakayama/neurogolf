
## Submit Gate

Default policy:

```text

medium risk: submit if gain >= 0.030, otherwise bank
high risk:   submit only if gain >= 0.080 and isolated
```

representation while preserving graph semantics. If gain is below `0.030`, bank
the exact candidate and skip submit.
