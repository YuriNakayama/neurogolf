# 20260702 case473 result - micro bundle submit cadence gate

## Decision

No Kaggle submit. Adopt the submit-gate infrastructure change.

## Change

Added an explicit micro bundle lane to the pre-submit gate:

```text
default: unchanged, low-gain bundles still block
opt-in flag: --allow-micro-bundle-gate
decision: submit-micro-bundle
allowed member decisions: bank-low-gain or submit-candidate
minimum aggregate gain: --micro-bundle-gain, default 0.015
maximum changed tasks: --micro-bundle-max-tasks, default 5
```

This does not weaken correctness:

```text
n_fail=0 is still required by each candidate gate
status=ok is still required
forbidden ops/functions still block
known review-risk or hard-risk tasks still block the micro lane
local-fail candidates still block
```
