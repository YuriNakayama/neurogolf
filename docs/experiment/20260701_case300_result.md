## Change

The original graph gated the corner block mask with scalar `hascorner`:

```text
hascorner = Greater(cornerc, hf)
cblk2     = And(cblk, hascorner)
s2        = Where(cblk2, corneru, s1)
```

`hascorner` is redundant at this consumer.  If `cblk` is empty, both `cblk` and
`cblk2` are false.  If `cblk` is nonempty, `cornerc > 0.5`, so `hascorner` is
true and `cblk2 == cblk`.

The adopted graph uses:

```text
s2 = Where(cblk, corneru, s1)
```

and removes the dead `hascorner`, `cblk2`, and `hf` initializer.
