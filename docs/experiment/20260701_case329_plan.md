## Hypothesis

The tail currently creates a 3-channel uint8 tensor with one explicit zero
channel, then pads to the final 10-channel output:

```text
Concat(ch0_valid, zero1_valid, ch2_valid) -> valid3
Pad(valid3, pads) -> output
```

Replace it with only the two live channels and a one-hot `QLinearConv`
projection:

```text
Concat(ch0_valid, ch2_valid) -> valid3_packed
QLinearConv(valid3_packed, one-hot 2ch->10ch weights, pads=[0,0,21,27])
  -> output
```

The projection maps the packed channels back to output channels 0 and 2 while
folding the final spatial pad into the convolution.
