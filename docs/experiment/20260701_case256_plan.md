## 仮説

`task112` は opset 13 で、`Concat(zero_4x6, motif_rev_rows, axis=1) -> top_band` がある。`zero_4x6` は `(4,6)` の all-zero bool initializer で、`motif_rev_rows` の左側 6 列 padding と等価。

`Pad(motif_rev_rows, pads=[0,6,0,0])` に置換すれば、24 要素の zero initializer を削り、4 要素の pads initializer を追加するだけで済む。memory は変わらないが params が約 20 下がる見込み。

## 手法

1. `Concat(zero_4x6, motif_rev_rows)` を `Pad(motif_rev_rows, pads=[0,6,0,0])` に置換する。
2. `zero_4x6` initializer を削除する。
3. `src.evaluate.scorer.audit_one` で `task112.onnx` のみ検証する。
4. `n_fail=0` かつ cost 減なら submit し、Public Score が上がれば採用する。
