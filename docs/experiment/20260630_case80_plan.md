# 20260630 case80 plan — t080 の bool→uint8 Cast を ReduceMin 後へ遅延できるか検証

## 仮説
t080 は cost 17136（params 112 / memory 17024）で、`lineOrOut` bool `[1,1,30,30]` を
uint8 に Cast してから `ReduceMin(axis=3)` している。
もし `ReduceMin(bool)` が許可されるなら、Cast を `[1,1,30,1]` まで遅延でき、約870B削減できる。

## 対象
- task080
- 現行: cost 17136, params 112, memory 17024

## 手法
1. `Cast(lineOrOut)->islineu` を削除。
2. `ReduceMin(lineOrOut)->rowline_b` を追加し、その後 `Cast(rowline_b)->rowlineu`。
3. `audit_one` で構造・正答性・cost を確認する。
