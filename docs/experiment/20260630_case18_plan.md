floor が multi-node net を使うタスクのうち、真に per-channel 線形分離なものは単一 Conv（hidden安全な
真アルゴリズム）で安く解ける。全 measurable multi-node タスクを 3×3/5×5 で網羅検定し、厳密分離する
ものがあれば単一 Conv 化して golf。

- 5×5-local（not 3×3）タスクの table サイズを確認（小さければ単一 5×5 Conv 候補）。
