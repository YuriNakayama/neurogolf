## Hypothesis

The tail currently constructs a sparse 10-channel small output with many
explicit zero channels, then pads to the final output:

```text
Pad(Sacc4, spad) -> Sg
Pad(Lmask, spad) -> Lg
Concat(black, zc, Lg, zc, zc, zc, zc, zc, Sg, zc) -> small
Pad(small, pads) -> output
```

Only three live channels are needed. Replace this with:

```text
Concat(black, Lg, Sg) -> packed3
QLinearConv(packed3, one-hot 3ch->10ch weights, pads=final output pad) -> output
```

The projection preserves the original output channel layout and removes seven
explicit zero-channel references from the final materialization.
