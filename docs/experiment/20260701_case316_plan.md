## Hypothesis

The recent high-transfer wins came from exact aliases that delete a whole local
subgraph. The next best candidates should have one of these shapes:

- same-shape aliases where a late predicate, mask, or reduction is equivalent to
  an earlier value and all dependent nodes become dead
- source-crop rewrites where upstream tensors shrink enough to overcome the
  extra slice parameters
- small exact cleanups that are below the submit gate but can be accumulated
  later in a multi-task bundle
