## Hypothesis

The tail currently builds an 8-channel 3x3 float16 tensor with six explicit zero
channels, then pads to the 10-channel 30x30 output:

```text
Concat(blue_patch, zero_patch x6, cyan_patch) -> output_3x3_c1_to_c8
Pad(output_3x3_c1_to_c8, pads=[0,1,0,0,0,1,27,27]) -> output
```

Only two live channels are needed. Replace this with:

```text
Concat(blue_patch, cyan_patch) -> packed_live
Conv(packed_live, one-hot float16 2ch->10ch weights, pads=[0,0,27,27]) -> output
```

The projection maps:

```text
output ch1 <- blue_patch
output ch8 <- cyan_patch
all other output channels <- 0
```

and performs the final bottom/right zero pad directly.
