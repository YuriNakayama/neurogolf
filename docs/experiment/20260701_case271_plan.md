## Target

## Hypothesis

`task157` has many all-zero bool initializers used only to pad local masks before later selection. Replacing those two-input zero `Concat` nodes with `Pad(dynamic, pads)` should remove zero initializer params. Unlike single micro-prunes, applying all eligible replacements in one task could save roughly `166` params before accounting for new pad initializers, making the relative reduction potentially large enough to matter.

## Method

1. Generate a temporary ONNX variant in `/tmp`.
2. For each two-input `Concat` with one all-zero initializer and one dynamic tensor:
   - infer axis and shape;
   - replace with `Pad(dynamic, pads)` when `arr.size - 2 * rank > 0`;
   - remove unused zero initializers.
3. Run `onnx.checker` and `src/evaluate.audit_one` only for `task157`.
4. If `n_fail=0` and cost improves, submit; otherwise reject without touching `data/output/onnx`.
