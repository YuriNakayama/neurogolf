## 仮説

## 検証
- `features_u8` の Concat から `ones_u8` を削除する。
- `final_w` の入力 channel 4本目を bias int32 に移し、weight を3 channelへ crop する。
- `src.evaluate.scorer.audit_one` で `task160` の全ペア完全一致と cost を確認する。
