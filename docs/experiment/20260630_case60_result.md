# 20260630 case60 result — t192 の 8-noise 規則は局所判定でなく rectangle 構造依存、安価構築不可（提出なし）

## independent_impl（必須欄）
case52 候補 t192(floor 8621, 16node, float 3916B)を decode:
- **変化は 8-cell のみ**（3↔0 の移動なし）。16 個の 8 → 9 個が 3, 7 個が 0 になる denoise。
- **手構築（局所規則探索）**: 8→{0,3} の判定を多数試行:
  - 4近傍 any-3, 3×3 any-3 → 0/265。
  - 3-count >= T (T=1..5) → 最良 2/265。
  - (3|8)-count >= T → 最良 1/265。
  - **全て不一致**。lut=11704(大)で、8→色 は**局所近傍では決まらない**。
- 8 は 3-矩形の上に乗ったノイズで、8→3 は「その 8 が 3-矩形の bounding 内か」= **矩形構造（大域）
  依存**。局所 k×k では判定できず、floor の 16-node が矩形検出を実装している。

## 確定
- t192 は **大域的矩形構造依存の denoise** で、局所規則(uint8 Conv)では安く解けない。
  floor 8621 は矩形検出を含む妥当なコスト。手構築で局所化を試みたが全失敗（最良 2/265）。
- 学習fit は lut=11704 を CNN で学べても hidden 過学習リスク(case1)＋大域文脈で深い net 必須→cost不利。

## 恒久方針の遵守
- 停止せず t192 を decode し局所規則不可を実証（手構築の失敗ログ記録）。floor 7180.78 退行なし。
- 「不可能の一般化」でなく、複数の局所規則を実 test した上での結論。次も継続。
